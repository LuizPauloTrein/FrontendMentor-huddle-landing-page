# 💬 Huddle Landing Page

Projeto desenvolvido como parte de um desafio do Frontend Mentor.

## 📌 Sobre o projeto

Este projeto consiste em uma landing page, com foco em apresentar um layout moderno e responsivo.

O objetivo principal foi praticar estruturação de páginas completas, uso de Flexbox para layout e construção de uma interface responsiva.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Variáveis CSS
- Media Queries (Responsividade)
- Google Fonts

---

## 🎯 O que eu pratiquei

- Estruturação semântica utilizando '<header>', '<main>', '<section>' e '<footer>'
- Criação de layout responsivo para mobile e desktop
- Organização de arquivos CSS (reset, variables, style, responsive)
- Uso de Flexbox para posicionamento dos elementos
- Estilização de botões e efeitos de hover
- Importação e aplicação de fontes do Google Fonts
- Implementação de layout baseado em design do desafio
